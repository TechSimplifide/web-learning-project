# Pokémon Image Search

A simple web app that lets users search for Pokémon images by name and view them instantly on the page.  
This tool is built with basic HTML, CSS, and JavaScript, and integrates Pokémon data to fetch and display images dynamically.

🔗 **Live Demo:** <a href="https://techsimplifide.github.io/pokemon-image-search/"> Deploy on GitHub </a>

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript

---

## 🚀 Features

- 🔍 Search Pokémon by name or id
- 🖼️ Display matching Pokémon image(s)
- Simple and clean UI
- Instant results without page reload

---

## 🧠 How It Works

1. The user enters a **Pokémon name** into the search input.
2. JavaScript handles the input and **fetches Pokémon image data**.
3. If a Pokémon with that name exists, its image is displayed on the page.
4. If not, an error message or fallback is shown.

This project demonstrates **DOM manipulation**, **event handling**, and optionally working with public Pokémon art sources or APIs (e.g., PokeAPI) to retrieve images and details. :contentReference[oaicite:0]{index=0}
