# 🌦️ Weather Vibe App

A modern, user-friendly weather web application that shows real-time weather information for any city.  
Built with HTML, CSS, and JavaScript, and powered by a weather API to deliver accurate conditions with a clean UI and smooth interaction.

🔗 **Live Demo:** <a href="https://techsimplifide.github.io/weather-vibe-app/">Deploy on GitHub </a>

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌤️ Current weather details including:
  - Temperature
  - Weather condition (clear, rain, clouds, etc.)
  - Humidity
  - Wind speed
- 📱 Responsive and clean design
- 💡 Smooth UI updates without page reloads

---

## 🛠️ Built With

- **HTML5** for markup
- **CSS3** for modern, responsive styling
- **JavaScript (ES6+)** for logic and API interaction
- **Weather API** for real-time data

## 📌 How It Works

- You enter a city name in the search input.
- JavaScript fetches weather data from a remote weather API.
- The app updates the interface with:
        - Temperature
        - Weather description
        - Weather icon
        - Additional details like humidity and wind
-Users immediately see updated weather results without refreshing the page.
