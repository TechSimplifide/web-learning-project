# BMI Calculator Web App

A simple and interactive **Body Mass Index (BMI) Calculator** web application that allows users to calculate their BMI based on their height and weight, and displays the BMI value along with a health category (underweight, normal, overweight).

🔗 **Live Demo:** <a href="https://techsimplifide.github.io/BMI-calculator-web-app/">Deploy on GitHub</a>

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript

---

## 💡 Features

- 🧍 Enter weight (kg)
- 📏 Enter height (cm / m)
- ➗ Calculates BMI instantly
- 📊 Displays BMI category:
  - Underweight
  - Normal weight
  - Overweight
- ✔️ Clean and responsive UI

---

## 🧠 How It Works

1. The user enters their **weight** and **height**.
2. Clicking the **Calculate** button triggers JavaScript logic.
3. BMI is calculated using the formula
4. The result and corresponding health category are displayed dynamically.
