# Number Guessing Game

A fun and simple web-based number guessing game where the user tries to guess a randomly generated number within a limited number of attempts.  
A friendly interface shows feedback on each guess and displays a win/lose message.

🔗 Live Demo: <a href="https://techsimplifide.github.io/number-guessing-game/"> Deploy on GitHub Pages</a>

## 🧠 How It Works

1. A random number is generated when the game starts.
2. The user enters a guess into the input.
3. Each guess is evaluated:
   - If the guess is too high or too low, feedback is shown.
   - If the guess is correct or the attempts run out, the game ends.
4. The user can restart the game without reloading the page.

---

## 🚀 Features

- Generates a random number each game
- Tracks user guesses and remaining attempts
- Provides user feedback (too high / too low / correct)
- Clean and responsive interface
