# Password Toggler

A simple and clean web app that lets users toggle the visibility of a password input.  
Clicking the eye icon will switch between showing and hiding the password.

🔗 Live Demo: https://techsimplifide.github.io/password-toggler/

## 🛠️ Built With

- HTML
- CSS
- JavaScript

```text
password-toggler/

├── script.js
└── README.md
```

---

## 🧠 How It Works

1. User types a password into the input field.
2. Clicking the eye icon toggles between:
   - showing the password (plain text)
   - hiding the password (dots/asterisks)
3. JavaScript listens for the click event and toggles input type (`password` ↔ `text`).
