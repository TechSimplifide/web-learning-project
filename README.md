# URL Shortener

A simple and responsive **URL Shortener** web application that allows users to generate a short link from a long URL.  
This project uses HTML, CSS, and JavaScript and integrates with a URL shortening API to generate clean, shareable short links instantly.

🔗 **Live Demo:** <a href="https://techsimplifide.github.io/URL-Shortener/"> Deploy on GitHub</a>

---

## 🚀 Features

- ✂️ Generate shortened links from long URLs
- 📋 Copy short link with a single click
- ⚡ Instant results without page reload
- 📱 Responsive and mobile-friendly design

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **URL Shortening API** (e.g., shrtco.de, TinyURL, or similar)

🧠 How It Works

- User enters a long URL in the input field.
- JavaScript sends the URL to a URL shortening API.
- The API returns a shortened link.
- The app displays the shortened URL.
- Users can copy the short link to the clipboard for easy sharing.
