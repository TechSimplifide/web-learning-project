# Login & Signup Form UI

A modern and responsive **Login & Signup Interface** built using HTML, CSS, and JavaScript.  
This elegant UI demonstrates smooth form transitions and validation for an enhanced user experience.

🔗 **Live Demo:** <a href="https://techsimplifide.github.io/login-signup-form-ui/"> Deploy on GitHub</a>

---

## ✨ Features

- 👤 Clean **Login form**
- ✍️ Interactive **Signup form**
- 🔄 Smooth transition between Login and Signup
- 👍 Responsive layout for mobile and desktop
- 💡 Basic client-side validation

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
