# 👤 Get Random People Profiles

A simple and beginner-friendly web app built using **HTML**, **CSS**, and **JavaScript**. It fetches and displays random user information (name, email, and photo) using the **Random User API**. This project is perfect to practice working with APIs and understanding `async/await` in JavaScript.

---

## 🧠 Features

- 🔁 Fetches a random user profile with one click
- 🧑‍💼 Displays user’s name, email, and profile picture
- ⚙️ Uses `async/await` for asynchronous fetch
- 📱 Responsive and clean layout
- 📦 No backend needed
